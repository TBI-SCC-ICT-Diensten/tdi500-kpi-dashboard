# 📸 Screenshots

Organiseer je screenshots in deze mappen:

- **scrumboard/** — dagelijkse scrumboard screenshots
- **features/** — werkende features in de browser
- **code/** — relevante code screenshots
- **tests/** — testresultaten screenshots
- **design/** — wireframes, diagrammen, flowcharts
- **git/** — PR's, branch history, commit logs
- **review/** — presentatie/review screenshots

> **Tip:** Gebruik duidelijke namen zoals `scrumboard_dag_1.png`, `feature_kpi_chart.png`, `test_api_happy_flow.png`
