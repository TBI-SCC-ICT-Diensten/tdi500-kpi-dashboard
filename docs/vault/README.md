# TDI 500 — Obsidian Vault

This folder is an Obsidian vault containing sprint workflow notes, daily logs, feature tickets, and test scaffolding for the BPV exam project.

## How to use

Open this folder directly in Obsidian via **File → Open folder as vault**. Wikilinks (`[[…]]`) resolve relative to this folder.

## What's in here

- **Daily/** — daily sprint notes (one per working day, 16 Mar – 27 Mar 2026)
- **Features/** — one note per GitHub issue / feature branch
- **Tests/** — test scenarios per feature
- **Templates/** — Obsidian templates for daily notes, feature notes, test notes
- **Screenshots/** — image attachments referenced from notes (scrumboard, code, features, etc.)
- **Exam Dashboard.md** — top-level sprint tracker
- **Definition of Done.md** — sprint DoD
- **Verbetervoorstellen.md**, **Retrospective.md**, **Overleg & Afspraken.md** — supporting notes

## Per-machine state

Obsidian's `workspace.json` (open tabs, panel layout) is gitignored — see the repo root `.gitignore`.
